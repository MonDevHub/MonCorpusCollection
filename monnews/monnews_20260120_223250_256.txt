သၟတ်လ္ၚဵု ဝေၚ်ဂေမ် Game ၜက်သၞောတ်တုဲ က္ဍဟ်လီုဒှ်ကၠုၚ်နွံဂး

သၟတ်လ္ၚဵုဂှ် ဝေၚ်ဂေမ် Game နကဵုဖုၚ် စောဲလာံၜက်သၞောတ်တုဲ က္ဍဟ်လီုဒှ်ကၠုၚ်နွံမံၚ်ရောၚ်ဂှ် မိမသၟတ် နူပွိုၚ်ဍုၚ်မုဟ်ဍုၚ် မွဲတၠဂှ်ကီု သီုကဵုညးချဳဒရာၚ်ပရေၚ်သၟတ်မန်မွဲတၠကီု ဟီုထ္ၜးရ။

သၟတ်တံဗွဲမဂၠိုၚ်ဂှ် ဝေၚ်ဂေမ် Game နကဵုဖုၚ် ဗီုပြၚ်လံၚ်ဩန်တုဲ သၟတ်လ္ၚဵုဂှ် စောဲလာံၜက်အာသၞောတ်တုဲ ဒ္ဂေတ်လီု က္ဍဟ်လီုဂှ် သၟတ်ဒေသမန်တံ စဒှ်ကၠုၚ်မံၚ်ညိညိယျရောၚ်ဂှ်လေဝ် ညးချဳဒရာၚ်ပရေၚ်သၟတ်မန် လတူဍုၚ်မတ်မလီုမွဲတၠ ဟီုသာ်ဝွံရ။

 သၟတ်တံဝေၚ်ဂေမ် Game တုဲ လ္ၚဵုဂှ် သ္ၚိဌာန်ဟွံကေတ် မိမဗက်ဂၠာဲဟွံဒး နွံကၠုၚ်မံၚ်ယျ၊ ဍေံတံ ဝေၚ်လံၚ်ကဵုဩန်ကဵု သ္ကအ်ရဲဍေံ ၜက်သၞောတ်တုဲ ဗီုကဵုစောဲလာမ်အာ၊ စောဲလာံ ၜက်သၞောတ်တှေ် ဗီုကဵုက္ဍဟ်လီု ဗီုယှောတ်ဒှ်အာ နွံမံၚ် ပ္ဍဲမန်ပိုဲစဆဵုမံၚ်ယျ သာ်ဝွံ ညးချဳဒရာၚ်ပရေၚ်သၟတ် လ္တူဍုၚ်မတ်မလီုမွဲတၠဂှ် ဟီုထ္ၜးရ။

သၟတ်စောဲလာမ်ဝေၚ် Game လ္ၚဵုဂှ် စောဲလာမ်ၜက်သၞောတ်တုဲ ဟွံစဴသ္ၚိဌာန်လေဝ်နွံ၊ ဗက်သ္ကအ်ရဲ ဟွံတီတမ်ကၞောတ်ကီု ဝိုတ်ပုၚ် ဝိုတ်သွ ဒှ်အာဗီုတမြအ် က္တဵူဒှ်ကၠုၚ်လေဝ်နွံကၠုၚ်ရ။

 တ္ၚဲတၟးတှေ် ဍေံသ္ပကမၠောန်ကဵု ဝေၚ်ဂေမ် Game ရ၊ ဍေံအာတ်လှဲဟွံဂွံ ဍေံထေဲ လအ်လအ်တှေ် သ္ပဟွံမာန်၊ ဍေံဟွံဂွံတှေ် စိုတ်ဍေံကြာန်တိုန်၊ အာ ဟွံကေတ်သ္ၚိရ၊ မုဟီုလေဝ် ဍေံဟွံကလၚ်ရ၊ ဍေံမံၚ်သၟးသၟးတှေ် ဗီုကဵုဍေံကူချပ်မံၚ် မွဲမွဲသာ် ၜက်အာနူသၞောတ်တှေ် ဒှ်အာဗီုမၞိဟ်က္ဍဟ်ဟွံပေၚ် မုလေဝ်ဟီုဟွံဂွံရ သာ်ဝွံ မိမသၟတ် နူပွိုၚ်ဍုၚ်မုဟ်ဍုၚ်မွဲတၠဂှ် ဟီုထ္ၜးရ။

သၟတ်ဒေသမန်ဂှ် သုၚ်စောဲဂဥုဲၜူမာဲၜက်သၞောတ်တုဲ ဝိဝိၚ်ဘိၚ်ဘၚ်ၜူမာဲဒၟံၚ် ကၠုၚ်မံၚ်ဂှ် လၟုဟ်မ္ဂး သၟတ်လ္ၚဵုတံ သီုစောဲလာမ် ဝေၚ် Game နကဵုဖုၚ်ဂှ် ဒှ်ကၠုၚ်ဗီုပြၚ်က္ဍဟ်လီုပဵု ကၠုၚ်မွဲသာ်ပၠန်တုဲ သွက်မိမယာဲတသၟတ်တံ တာလျိုၚ် ဇၞော်ကၠုၚ်မွဲသာ်ပၠန်ရ။

 ဍေံ (သၟတ်) တံဝေၚ်မံၚ်ဂှ်တှေ် ဍေံညးမွဲဓဝ် ကဵုဖုၚ် (တၠဂေမ်) လေဝ်နွံ၊ ဝေၚ်ကဵုဂကောံလေဝ်နွံ၊ ဍေံတံ စုတ်လှဲတုဲတှေ် ကျတှေ် ဒးကဵုလှုဲ၊ ဇၞးတှေ် ဂွံလှုဲ၊ လ္ၚဵုတှေ် ကျ ဇၞး ကဵုလက်ပြၚ်နွံ၊ ဝေၚ်လှုဲဂၠိုၚ်ဂၠိုၚ်ဂှ် ဆဵုအောန်၊ ဆဂးတှေ် ဝေၚ်တုဲ စောဲမံၚ်၊ လ္ၚဵုဂှ်တှေ် ဝေၚ် ဟွံကေတ်ကေမၠာန်လေဝ်နွံ၊ လ္ၚဵုဂှ် စောဲအာၜက်သၞောတ်တုဲ ဗီုကဵု ဝိဝိၚ် စိုတ်ယောၚ်အာလေဝ်နွံ၊ ဝိုၚ်ပိုဲတှေ် ရဲဝေၚ်ဂေမ် Game တံ ၂၄ နာဍဳရ ပြၚ်မံၚ်လှုဲ သာ်ဝွံ မိဣလောန် တၠဆေၚ်ပြၚ်ပလံၚ်ဩန် နကဵုကေပေ Kpay နူဍုၚ်ဇြပ်ဗုဂှ် ဟီုရ။

သၟတ်စောဲလာမ်ဝေၚ်ဂေမ် Game ၜက်သၞောတ်လ္ၚဵုလ္ၚဵုတံဂှ် စိုတ်ဒှ်အာဗီုတမြအ်မွဲတုဲ ဟွံကေတ်သ္ၚိဌာန်ဒှ်ကၠုၚ်ကီု၊ မိမယာဲတဟီု ဟွံကလၚ်ဒှ်ကၠုၚ်လေဝ်နွံကီု၊ လ္ၚဵုဂှ် ဟီုဟွံကေတ် ဝေတ်ဟွံဒုၚ် မိမတံ သက်သ္ပဟွံမွဲတုဲ ဒဵုကဵုဒးချပ်ကဵု ဗော်ဍုၚ်မန်တၟိ ထိၚ်ဒဝ်ရပ်စပ်လေဝ် နွံကၠုၚ်မံၚ်ဂှ် ဂွံတီကေတ်ရ။
