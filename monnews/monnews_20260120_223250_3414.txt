ကၠောန်ဗဒှ်အာ သဘၚ်သဳကၠဳ ပေန်လုၚ်ရောၚ် ဗော်မန်ဒဳမဝ်ဟီု

နာဲထဝ်သောန်

စၠန်ထဝ် 

ပံၚ်တောဲကဵု ဒဝ်အံၚ်သာန်သုကျဳတုဲ သွက်ဂွံပၠုပ်လမျီု ကၟိန်ဍုၚ်ပံၚ်ကောံဖေတ်ဒရေဝ်ဂှ် ကၠောန်ဗဒှ်အာ သဘၚ်သဳကၠဳပေန်လုၚ် ဒုတိယရောၚ် ဥက္ကဌ ဗော်မန်ဒဳမဝ် နာဲထဝ်သောန်

(

နာဲထောန်တိၚ်

)

ကာလဆဵုဂဗကဵု ဒဝ်အံၚ်သာန်သုကျဳဂှ် ဟီုဂးလဝ် နွံရ။

ဒဒှ်နာဲထဝ်သောန် ဟီုဂးလဝ်ဂှ် နာယက ကၠောန်ဗဒှ် သဘၚ်တ္ၚဲကောန်ဂကူ နာဲတေန်အံၚ် ကလေၚ်ဟီုလဴထ္ၜးကဵု ပ္ဍဲပေါဲကောံဓရီု ပံက်ကမ္မတဳကၠောန်ဗဒှ် သဘၚ်တ္ၚဲကောန်ဂကူမန် ပ္ဍဲဂိတုနဝ်ဝေမ်ပါ

(

၁၇

)

လောန်ကၠုၚ်ဏံရောၚ် ညးတိုန်ကလၚ် တိုန်စိုပ်လဝ်မွဲတၠ သၟတ်ဗြဴရဲမုဟ်ဍုၚ် ဟီုရ။

ပံၚ်ကောံကဵု ဂကောံကောန်ဍုၚ်အရၚ်တံတုဲ ဒက်ပ္တန်အာ ကၟိန်ဍုၚ်ဖေတ်ဒရေဝ်မွဲရောၚ်ဂှ် ဟီုဂးဏာလဝ် ပ္ဍဲကောံဓရီုရ သာ်ဝွံ သၟတ်ဗြဴတိုန်စိုပ်လဝ် ကောံဓရီုဂှ် ဟီုရ။

အပ္ဍဲပွဳပွိုၚ်တ္ၚဲတေံပၠန် သွက်ဂွံကၠောန်ဗဒှ် သဘၚ်သဳကၠဳပေန်လုၚ် ဒုတိယဂှ် က္ဍိုပ်သ္ကိုပ်ကောန်ဍုၚ်အရၚ်ဂမၠိုၚ် ဆဵုဂဗကဵု ဒဝ်အံၚ်သာန်သုကျဳ ဒၞာဲရုၚ်

NLD

ဍုၚ်လ္ဂုၚ်တေံကီု၊ အပ္ဍဲက္ဍိုပ်သ္ကိုပ်တံဂှ် နူကဵုဗော်မန်ဒဳမဝ် ဥက္ကဌ နာဲထဝ်သောန် ကေုာံ ဒုဥက္ကဌ နာဲၚိုဲတိၚ်တံလေဝ် တိုန်စိုပ်လဝ်ကီုရောၚ်ဂှ် ဂွံတီကေတ် နူဗွိုၚ်ညးတံမွဲတၠရ။

သွက်ဂွံကော်ဘိက် သဘၚ်သဳကၠဳပေန်လုၚ် ဒုတိယမာန် ပ္ဍဲဂိတုအံက်တဝ်ပါ

(

၂၄

)

ပတိတ်လဝ် လိခ်လလောၚ်တြး ဍုၚ်ကလေဝ် ဂှ် က္ဍိုပ်သ္ကိုပ်ဝှ်ကြံၚ် ဂကောံ

NLD

တံသၟးဟွံက နူဗော်မန်ဒဳမဝ် နာဲထဝ်သောန် ကေုာံ နာဲၚိုဲတိၚ်တံလေဝ် တုပ်စိုတ် ထပက်လဝ်စၟတ်တဲကီုရ။

တၞဟ်ပၠန် ဟိုတ်နူဒဝ်အံၚ်သာန်သုကျဳ ဗၠးၜးတိတ်ကၠုၚ်သာ်ဏံဂှ် ဗီုဂကောံကောန်ဍုၚ်တၞဟ်ဟ်ကီု ဗော်ဍုၚ်မန်တၟိလေဝ် ကဵုဏာလိခ်သဝဏ် ဗွဲမမိပ်စိုတ်ကီု ဂွံတီရ။

ပေါဲကောံဓရီု ကမ္မတဳဗဟဵု ကၠောန်ဗဒှ် သဘၚ်တ္ၚဲကောန်ဂကူမန်ဂှ် ကၠောန်ဗဒှ် ဒၞာဲဇြပ်သဘၚ်ဓရ်မန် ဍုၚ်မတ်မလီုတုဲ နူပွိုၚ်ဍုၚ်

(

၈

)

ပွိုၚ်ဍုၚ် မၞုံအပ္ဍဲတွဵုရးဍုၚ်မန်ဂှ် ညးစၞးၜိုတ်

(

၁၀၀

)

တၠ တိုန်စိုပ်လဝ် ဂွံတီရ။ ပ္ဍဲတ္ၚဲဂှ်ဟေၚ် ရုဲစှ်သ္ပဏာဒတန် ဂကောံကမ္မတဳဗဟဵု ကၠောန်ဗဒှ် သဘၚ်တ္ၚဲကောန်ဂကူမန်

(

၃၀

)

တၠ တုဲဒှ်အာရောၚ် ဂွံတီကေတ်ရ။
