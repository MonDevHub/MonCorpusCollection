ကဵု ဒပ်ကရေၚ် ဒဳကေဗဳအေ

(

၉၀၆

၉၀၇

)

တအ် ပံၚ်တုဲ ဇြဟတ်ဍေံ နွံၜိုတ်

(

၄၀၀

)

တံဂှ်၊ ဍေံတအ်ဂေတ်မံၚ် ဗွိုၚ်၂၄ ကွာန် ပ္ဍဲဒေသ ခရိုၚ်မတ်မလီုဏအ်ရ သာ်ဝွံ ကောန်ကွာန်သၞေဟ်သြောံ အပ္ဍဲဒေသခရိုၚ်မတ်မလီုမွဲတၠ ဟီုရ။

အတိုၚ်ညးလ္ၚတ်ပရေၚ်ပၞာန်မွဲတၠဟီုဂှ် ဒပ်ပၞာန်အလဵုအသဳဗၟာတအ် စှေ်ထ္ပက်ပၞာန် ဝေၚ်မံၚ်ပၞာန် သကုတ်ဒေသခရိုၚ်မတ်မလီုလၟုဟ်ဏအ်ဂှ် အစဳအဇန်ဍေံတအ် ထ္မံက်ထ္ၜးလဝ်ဂှ် သွက်ဂွံထ္ပက်ပၞာန်ကဵု ဒပ်ပၞာန်ကရေၚ် ကေအေန်ယူ

(

တပ်မဟာ ၆

)

နွံ ဆဂး လၟုဟ်ဏအ်ဂှ် ဒပ်ပၞာန်အလဵုအသဳတအ်ဂှ် ဂေတ်မံၚ် သကုတ်ဒေသဗွိုၚ်တိဍာ် ဗော်ဍုၚ်မန်တၟိ ခရိုၚ်မတ်မလီုဏောၚ် ။

အကာဲအရာပြဟ်ပြဟ်ဏအ်ဂှ် ပၞာန်ဗၟာတံ လ္တူဗော်ဍုၚ်မန်တၟိတံဂှ် ပရေၚ်ခက်ခုဲမုလေဝ် ကၠောန်သ္ပကၠုၚ် ဟွံမွဲဏီပုဟ် အကာဲအရာဂတတေံ ဗီုလဵုကၠုၚ်ပၠန်ဂှ်လေဝ် နကဵုပိုဲတံ ဆက်လ္ၚတ်မံၚ်ဖိုဟ်ရောၚ် ညးဆက်ဟီုရ။

အခိၚ်ဒပ်ပၞာန်ဗၟာတအ် စှေ်ထ္ပက်မံၚ်ပၞာန် သကုတ်ဒေသခရိုၚ်မတ်မလီုဂှ် ညးဍုၚ်ကွာန် ဒေသပွဳူတအ် ဒှ်ဓဝိၚ် ဖေက်ဒပ်ပၞာန်ဗၟာတအ် ရပ်တံၚ်ဗဝ်တာကွေံကွေံရောၚ်။ နူကဵု ပၞာန်ဗၟာတအ် ယဝ်ဒှ်ဝေၚ်ပၞာန်ကဵု ကေအေန်ယူတအ်တှေ် သွက်ဍေံတံ ဂွံကေတ်အရီုဗၚ် ကောန်ဗဝ်တာ နူညးဍုၚ်ကွာန်တအ်နွံမံၚ်၊ သ္ကိုပ်ဗျူဟာ နူကဵုဒပ်ဗၟာ စကခ

(

၇၇

)

ကဵုလဝ်စၞောန်ကဵု ဥက္ကဌကွာန်တံနွံ သာ်ဝွံ မၞိဟ်စုတ်ကဵု ဥက္ကဌကွာန် အပ္ဍဲဒေသပွဳူ ခရိုၚ်မတ်မလီုမွဲတၠ ဟီုရ။

လောန်ကၠုၚ် ဂျူလာၚ် ၁၂ တေံလေဝ် ဒေသကျာ်ပိ နူဝၚ်ကနာၚ် စဵုကဵုစိုပ်ကျာ်ပိဂှ် နူဒပ်ပၞာန်ဗၟာ ခမရ
