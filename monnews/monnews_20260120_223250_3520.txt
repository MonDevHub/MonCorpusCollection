ပေါဲရုဲမာဲ ၂၀၁၀ တုဲ ပြံၚ်သၠာဲမာန်ဟာ

တ္ၚဲဏံဂှ် အလဵုအသဳတၠအဝဵုပၞာန်တအ်ဂှ် ကဵုဏာသတိပါၚ် သွက်ဂွံကၠောန်သ္ပကဵု ပေဲါရုဲမာဲသၟးဟွံသေၚ်၊ သီုကဵုအခေါၚ်ကဵု ဗော်ပရေၚ်ဍုၚ်ကွာန်လ္ၚဵုတံ ပါလုပ်ပ္ဍဲကဵုရုၚ်မာဲဏအ်တုဲရ။ အတိုၚ်အလဵုအသဳပၞာန်တအ် ကၠောန်သ္ပမံၚ်ဏံ ရံၚ်အာဗွဲအပြောံမ္ဂး၊ နကဵု ဍေံတအ် ကၠောန်သ္ပပေဲါရုဲမာဲဏအ်တုဲမ္ဂး ဂၠံၚ်တရဴပရေၚ်ဍုၚ်ကွာန်၊ ဂၠံၚ်တရဴ ပရေၚ်ဒဳမဝ်ကရေဇြဳ ခိုဟ်တၟိုဟ်တိုန်ညိဟာဟွံတီ ညးဗွဲမဂၠိုၚ်တံ စၟီဒၟံၚ်အိုတ်ရ။

နကဵုအလဵုအသဳတအ်ဂှ် ဖျေံလဝ်အစဳအဇန် သွက်ဂွံထိၚ်ဒဝ်အာ အဝဵုဍေံတအ် ပ္ဍဲသၞာံဂတ

(

၂၀၁၀

)

ဏအ် မွဲနဲမွဲဗီုတုဲ အဝဵုအုပ်ဓုပ်ဍုၚ် ညံၚ်ဟွံဗၠးတိတ် နူကဵုသကေံတဲဍေံတအ်ဂှ် ဍေံတအ်ဂစာန်မံၚ်လ္ဂဵုရောၚ်။ လၟုဟ်လေဝ် သက်သဳသာဓကဍေံ နွံဒၟံၚ်တၟာဂလိုၚ်ရောၚ်။

ပေဲါရုဲမာဲဏအ်ဂှ် ဒုၚ်လဝ်သဇိုၚ်ကဵု သၞောတ်ဝ်ဒက်ပ္တန်ဍုၚ်

(

၂၀၀၈

)

တေံတုဲ သ္ကိုပ်ပၞာန်တအ်ဂှ် ဟွံဒးၜးကယျိုၚ်မွဲသာ်၊ တိုန်ကေတ်လဝ် လ္တူဒၞာဲခုၚ်အဝဵု ဗၞတ်

(

၂၅

%)

ဂွံရ။ လၟုဟ်ပြဟ်ဟ်ဏံ သ္ကိုပ်ပၞာန်တအ်ဂှ် ရာံတိတ်နူဘဝကောန်ပၞာန်တုဲ ဒက်ပ္တိုန်ဗော်ကျဖောအ်

(

ကြံ့ခိုၚ်ရေးပါတီ

)

ပလေဝ်လဝ် သွက်ဂွံလုပ်ပေဲါရုဲမာဲရ။ အခိၚ်ဗော်တၞဟ်ဟ်ဂမၠိုၚ်

(

သီုကဵုဗော် ကောန်ဍုၚ်အရၚ်

)

အဃောဒးရုန်ဂစာန်ဒၟံၚ် သွက်ဂွံပ္တိုန် စရၚ်ဗော်၊ ဂွံကလိ ဂွံသြန်ဘဏ္ဍာဂှ် နူကဵုဗော်ကျဖောအ်တအ် ကၠောန်သ္ပဒၟံၚ်ပရေၚ်စန်ဒက် ပ္ဍဲဒေသနာနာ၊ ပွိုၚ်ဍုၚ်နာနာတုဲ ဂွံအခေါၚ်နူအလဵုအသဳ သွက်ဂွံပကောံ သြန်ဘဏ္ဍာတအ်ရ။

ရံၚ်အာလ္တူအကာဲအရာဏအ်မ္ဂး အလဵုအသဳပၞာန်တ္ၚဲဏအ်ဂှ် ပေဲါရုဲမာဲ

(

၂၀၁၀

)

တုဲမ္ဂး နကဵုယၟုတၟိ သွက်ဂွံထိၚ်စွံလဝ်အဝဵု ဍေံတအ်ဂှ် ဖျေံလဝ်ဗီုရောၚ်။ ပ္ဍဲကဵုပေဲါရုဲမာဲဏံ ရန်ကဵုဗော်ဒဳမဝ်ကရေဇြဳကောန်ဂကူ

(

အေန်အေဝ်ဒဳ

)

ညံၚ်ဟွံဂွံပါလုပ်မွဲဝါပၠန်တုဲ စဵုဒၞာဲနဲနာနာသာ်ရ။ ဒဝ်အံၚ်သာန်သုကျဳဂှ် ဒးဒုၚ်ထိၚ်စွံလဝ် ပ္ဍဲကဵုသ္ၚိတုဲ က္ဍိုပ်သ္ကိုပ်ဗော်ဂမၠိုၚ်တအ် ဒးဒုၚ်ပၠုပ်လဝ်ထံၚ်ရ။

ဟီုသောၚ်ၚ်မ္ဂး ပ္ဍဲကဵုပေဲါရုဲမာဲဏအ်ဂှ် ဗော်ကျဖောအ်မွဲဗော်ဓဝ် ဍေံတအ်ကေတ်စေဝ်လ္တူတုဲ

ြ

ပိုၚ်ပကာန်ကဵု ဗော်ဍောတ်တ်တၞဟ်ဂမၠိုၚ်၊ ဍေံရန်လဝ် သွက်ဂွံဇၞး လ္တူဗော်တၞဟ်ဟ်တုဲ သွက်ဂွံဒက်ခၞံပ္တိုန် အလဵုအသဳအုပ်ဓုပ်ဍုၚ်ရောၚ်။ ယဝ်ရ အလဵုအသဳပၞာန်တအ် ကလေၚ်ဂွံအဝဵုမွဲနဲပၠန်မ္ဂး၊ ဗီုပြၚ်အုပ်ဓုပ်ဂှ် သှေ်ဒၟံၚ်အတိုၚ်လၟုဟ်ပၠန်ရောၚ်။

ပ္ဍဲကဵုဍုၚ်ဗၟာဝွံ ယဝ်ရအလဵုအသဳပၞာန်ဟေၚ်၊ ကလေၚ်ဂွံအဝဵုမွဲဝါပၠန်တုဲ ပေဲါရုဲမာဲဏအ် ဟွံတပ်တးမ္ဂး နူကဵုဍုၚ်မ္ၚးတအ် နကဵုပရေၚ်ဍုၚ်ကွာန်၊ ပရေၚ်ပိုန်ဒြပ် ကေုာံ ပရေၚ်မၞိဟ် သွက်ဂွံကဵုအရီုအဗၚ် ကုအလဵုအသဳဂှ် ဒှ်ဟွံမာန်ရ။ ရဲဓလိုက်တမ်ရိုဟ်တအ် ဒးစွံသတိဇၞော်ဇၞော် ပ္ဍဲအရာကၠောန်ပရေၚ်ပိုန်ဒြပ် သွက်အလဵုအသဳတၟိဝွံရ။

ကှေ် မုပြံၚ်သၠာဲမာန်ရော၊ ၜိုန်ရ ပ္ဍဲကဵုသၞာံ

(

၂၀၁၀

)

ဏအ် ပေဲါရုဲမာဲတုဲဒှ်အာကီုလေဝ် တၟေၚ်တၟဟ် စပ်ကဵုပရေၚ်သၞောတ် ပရေၚ်ဍုၚ်ကွာန်လေဝ် ဟွံပြံၚ်သၠာဲ၊ ပရေၚ်ပိုန်ဒြပ်ဟွံတဴတက်တိုန်၊ ပရေၚ်မၞိဟ် အဓိက ပရေၚ်ပညာကဵု ထတ်ယုက် ဟွံတဴတက်တိုန်မ္ဂး ဘဝညးဍုၚ်ကွာန်တအ်ဂှ် ပေဲါရုဲမာဲတုဲလေဝ် အတိုၚ်နူကဏေအ်တေံပၠန်ရရောၚ်။
