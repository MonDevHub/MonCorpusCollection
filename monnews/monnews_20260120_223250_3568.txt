ပရေၚ်ၜိုဟ်လလံ လအ်အာသ္ၚောဲအာ

ဂကူကောန်ဍုၚ်အရၚ်ဂမၠိုၚ် သီုကဵုကောန်ဂကူမန်ကီု ရန်ကဵုအခေါၚ်အရာ ကံကုသဵုဂကူဇကု ဇကုဂွံဖန်က်ကေတ်မာန်တုဲ လုပ်ကၠုၚ်ပေါဲဗတိုက် ပ္ဍဲဗွိုၚ် ပၠန်ဂတးကီု ပ္ဍဲကဵုဗွိုၚ်ဥပဒေကီုဂှ် လၟုဟ်မ္ဂး လအ်ကၠုၚ်ဗၞတ်

(

၆၀

)

သၞာံရ။ လောန်ကၠုၚ်ဗၞတ်

(

၁၅

)

သၞာံတေံ ဗော်ရပ်လွဟ်ကောန်ဍုၚ်အရၚ် ဗွဲမဂၠိုၚ်တအ်ကဵု အလဵုအသဳပၞာန်တအ် ဒေံါကၠုၚ်ပန်ပှော်တုဲ ကောန်ဍုၚ်အရၚ်တအ် စၟဳကၠုၚ်ပရေၚ် ၜိုဟ်လလံရ။

ဆဂး ပ္ဍဲအခိၚ်ခြာဟွံလအ်ကၠုၚ်တေံ အလဵုအသဳပၞာန်တအ် ကၟာတ်ထောံဍာန် ပရေၚ်ၜိုဟ်လလံရ။ ပ္ဍဲသၞာံ ၂၀၀၄ အဃောကောံဓရီု အလုံကောန်ဂကူ

(

အမျိုးသားညီလာခံ

)

ကီု အဃောသက္ကရာဇ် ၂၀၀၈ သ္ပဒတန်သၞောတ်ဝ် ဒက်ပ္တန်ဍုၚ်ကီု၊ အလဵုအသဳပၞာန်တအ်ဂှ် ဟွံတော်စုတ် အခေါၚ်အရာကောန်ဍုၚ်အရၚ်ရ။ ပၟိက်မိက်ဂွံသောၚ်ကလးပြသၞာ အပ္ဍဲဍုၚ် ဟွံမွဲရ။ ဍေံတအ်ဂွံအဝဵုတှေ်တုဲ ညးတၞဟ်တအ်မုဒှ်ဒှ် ချပ်ကၠုၚ်သာ်ဂှ်တုဲ ပရေၚ်ၜိုဟ်လလံဂှ် လအ်အာသ္ၚောဲအာရောၚ်။

ခြာဟွံလအ်ကၠုၚ်တေံ အလဵုအသဳပၞာန်ဗၟာတအ် ဓရတ်ကၠုၚ်ကောန်ဍုၚ်အရၚ် သွက်ဂွံပြံၚ်သၠာဲထောံ ဒပ်ပၞာန်ညးတအ် နဒဒှ်ဒပ်မၚ်ပယျဵု

(

ဝါ

)

ဒပ်ညးဍုၚ်ကွာန်

(

ပျဳသုသေတ်

)

တအ်ဂှ် ဍေံတအ်ကော်ဘိက်ပၞာန် အပ္ဍဲဍုၚ်တၟိရောၚ်။ သွက်ကောန်ဍုၚ်အရၚ်တအ်ဝွံ မုအခေါၚ်အရာညးတအ်ဟွံမွဲမွဲသာ် ညးတအ် ဂွံပြံၚ်သၠာဲထောံ ဒပ်ပၞာန်ပၠန်ဂတးညးတအ်ဂှ် ဝါတ်ရ။

ကောန်ဂကူမန်၊ ကချၚ်၊ ဝ၊ သေံဇၞော်တအ် ပၠန်ဂတးကၠုၚ်အလဵု အသဳပၞာန် အခိၚ်အခါမလအ်လကီု၊ လက္ကရဴဏအ် ဒေံါပန်ပှော်ကု အလဵုအသဳပၞာန်ကီုဂှ် ညးတအ်နွံပၟိက်ကဵုတိဍာ်ညးတအ် ညးတအ်ဟေၚ် မိက်ဂွံ အုပ်ဓုပ်ကေတ်တုဲ ကံကုသဵုဂကူညးတအ် နူကဵုဂကူဗၟာ လုပ်ဖန်က်ဒၟံၚ်ဂှ် ဟွံမွဲပၟိက်သက်က်ရ။ ဂကူကရေၚ်၊ ကရေၚ်ဍာဲကဵု ချေၚ်တအ် လုပ်ဒၟံၚ်ပေါဲဗတိုက် ပၠန်ဂတးဒၟံၚ်အခိၚ်တ္ၚဲဏအ်ဟီုဂှ် ကံကုသဵုညးတအ် ညးတအ် မိက်ဂွံဖန်က်ကေတ်ဟေၚ်ရောၚ်။ ပ္ဍဲကဵုလညာတ် ဆန္ဒကောန်ဍုၚ်အရၚ်တအ် ဖအိုတ်ဂှ် မိက်ဂွံဗလးတိတ်နူအဝဵုဂကူဗၟာ

(

ဝါ

)

ပၞာန်ဗၟာတုဲ ဒက်ခၞံပ္တိုန်ကၟိန်ဍုၚ် ပံၚ်တောဲတုဲ ဍုၚ်ညးကဵုညး ညးတအ်မိက်ဂွံအုပ်ဓုပ်ကေတ်ရ။

လၟုဟ်ဝွံ ဂကူဗၟာ

(

ဝါ

)

အလဵုအသဳပၞာန်တအ်ဂှ် သိုၚ်ခၞံလဝ်အဝဵုဍေံတအ် နကဵုပၠံၚ်လွဟ်တုဲ သီကေတ်တိဍာ်ကောန်ဍုၚ်အရၚ်တအ် ဂၠံက်ပ္တိတ်သွံ သံယံဇာတ ကောန်ဍုၚ်အရၚ်တအ်တုဲ ကလေၚ်ဗတိုက်ကောန်ဍုၚ်အရၚ် ထဍေၚ်ပ္တိတ်ကောန်ဍုၚ်အရၚ် ညးဍုၚ်ကွာန်ဓမ္မတာတအ်ရ။ မွဲအခိၚ်ဓဝ် ကၟာတ်လဒဵုလဝ်အခေါၚ် လိက်ပတ်ယေန်သၞာၚ် ကောန်ဍုၚ်အရၚ်တအ်အိုဿီု ဂစာန်ၚိတ်ကမၠက်ဒၟံၚ်ကောန်ဍုၚ်အရၚ်တအ် သွက်ဂွံဒှ်အာ ဂကူဗၟာရ။

ပ္ဍဲကဵုပွိုၚ်

(

၅၀

/

၆၀

)

သၞာံ ကောန်ဍုၚ်အရၚ်တအ် ပါလုပ်ကၠုၚ်ပရေၚ်ဍုၚ်ကွာန်၊ တဝ်စၞေဟ်ကၠုၚ် နကဵုပရေၚ်ပၠန်ဂတးတုဲ ပ္ဍဲကောန်ဍုၚ်အရၚ်တအ် လညာတ်ပရေၚ်ဍုၚ်ကွာန်၊ ဍာ်ဒကေဝ်ပရေၚ်ပၞာန် နွံကၠုၚ်ရ။ ကောန်ဍုၚ်အရၚ် လ္တူဒဵုကီု ပ္ဍဲတိဗလးကီု၊

ဝေအ်ဝေအ်ဝိုက်က်

ဟွံသေၚ်ရ။

ညးမအုပ်ဓုပ် ဂွံလဝ်အဝဵုပတေၚ်မ္ဂး ကောန်ဍုၚ်အရၚ်တအ် ယှအ်ဟွံသေၚ်ရ။ ပေါဲပၞာန်တၟိ ပ္ဍဲဂှ် သွက်ဂွမကလေၚ်ကတဵုဒှ်နွံရ။ အလုံကောန် ဍုၚ်ဗၟာကီု၊ အလုံကောန်ဍုၚ်အရၚ်ကီု စၟဳဒၟံၚ်ပရေၚ်ၜိုဟ်လလံဂှ် လအ်အာ သ္ၚောဲအာရ။
