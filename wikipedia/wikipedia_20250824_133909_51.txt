ဤတမ်းပလိတ်သည် {{Infobox election}} ၏
တမ်းပလိတ်ခွဲ တစ်ခုဖြစ်ပြီး တိုက်ရိုက်အသုံးပြုရန် မဟုတ်ပါ။
This subtemplate takes two inputs,
name and link.
If link is set to no,
then it returns the value of name verbatim.
Otherwise,
if the corresponding ထာမ်ပလိက်:
Tld template exists,
it returns [[{{{name}}}|shortname]] where shortname is the output of the ထာမ်ပလိက်:
Tld template.
If this template does not exist,
it returns the value of name verbatim.
