လဒက်ပ္တန် ဂၞန်သင်္ကေတ (Code System) သွက်ဍာ်ဇမၠိင်

ပ္ဍဲအရာမဒက်ပ္တန် ဂကောံအုပ်ဓုပ် ဟွံသေင်မ္ဂး သၞောတ်ထိင်ဒဝ်စရင်အင်မွဲမွဲ (ဥပမာ - စရင်သင်လုပ်ဒ္ဂိုန်) ဂှ်၊ ပရေင်ဖန်ဗဒှ်

ဂၞန်သင်္ကေတ (Code)

ဝွံ ဒှ်အရာ မကိစ္စဇၞော်အိုတ်ရ။ သွက်ဂွံ ရေင်တၠုင်အာ ဍာ်ဇမၠိင် (Long-term sustainability) မ္ဂး ဂၞန် Code တအ်ဂှ် သ္ဒးနွံကဵု သၞောတ်ဝ် မစှ်ေကဵု လၟေင်ကဆံင် ကေုာံ အဓိပ္ပါယ် မပၞုက်လဝ် ဍိုက်ဍိုက်ပေင်ပေင် ရ။

အရာသ္ဒးကူချပ် အခိင်မဖျေဟ်သၞောတ် Code

အခိင်ကာလ မစဖျေဟ် သၞောတ် Code မွဲမွဲမ္ဂး တင်အဓိက သ္ဒးကူချပ် သွက်ဍာ်ဇမၠိင်ဂှ် နွံ (၄) သာ်ရ-

ပရေင်သၠဲလး သွက်အနာဂတ် (Scalability)

- ပ္ဍဲအနာဂတ် ယဝ်ရ ထပ်တိတ်ကၠုင် ပွိုင်ဍုင်တၟိ၊ ဟွံသေင်မ္ဂး ဒေသတၟိမ္ဂး သွက်ဂွံ ထပ်စုတ် ဂၞန်တၟိဂှ် သ္ဒးဖျေဟ်လဝ် ဒၞာဲသၠး (Gap) မာန်ရ။ (ဥပမာ - စၞးမစုတ် ဂၞန်ဆက်က် ၁၇၊ ၁၈၊ ၁၉ ဂှ်၊ သ္ဒးသၠးလဝ် စဵုကဵု ၂၀ တုဲ၊ ခရိုင်တၟိမ္ဂး စကဵု ၂၁ ဒှ်တမ်)။

စှ်ေသၞောတ် ကေုာံ ဟွံထပ်ညးသ္ကအ် (Uniqueness)

- မွဲဒေသ၊ မွဲပွိုင်ဍုင် ဟွံသေင်မ္ဂး မွဲဘာမ္ဂး ဂၞန် Code နွံ ဆမွဲဟေင်ရ။ ဟွံကဵု အခေါင် ထပ်ညးသ္ကအ် သွက်ဂွံ ဝေင်ပါဲ ပရေင်ယောင်ယာ တင်ဂၞင် (Data Duplication) ရ။

လၟေင်ကဆံင် (Hierarchy)

- ဂၞန် Code ဂှ် သ္ဒးထ္ၜးကဵု ကဆံင်အုပ်ဓုပ်မာန်ရ။ (ဥပမာ - နူ ကဆံင်တွဵုရး စှ်ေအာ ကဆံင်ခရိုင်၊ နူဂှ် စှ်ေအာ ကဆံင်ပွိုင်ဍုင်)။

လောဲသွာ ကေုာံ ကၠိုဟ်လောဲ (Simplicity)

- ဂၞန်ဂၠိင်လောန်မ္ဂး ညးရပ်စပ်တအ် စၟတ်ဝါတ်၊ ချူဗၠေတ်မာန် တုဲ၊ ဂၠိလောန်မ္ဂး လၟေင်ကဆံင် ဟွံဍိုက်ပေင် ဒှ်မာန်ရ။ ဟိုတ်ဂှ်ရ သ္ဒးစကာ ဂၞန် မၞုံကဵု ဗၞတ်ဗ္ၜတ် (ဥပမာ - ၅ မ ဟွံသေင်မ္ဂး ၆ မ) ရ။

အဓိပ္ပါယ် မပၞုက်လဝ် ပ္ဍဲဂၞန် Code (Semantic Structure)

သၞောတ် Code မခိုဟ် သွက်ဍာ်ဇမၠိင်မ္ဂး၊ ညးမရံင်စရင်တအ် ဆဵုဂၞန် Code မ္ဂး တီကေတ် ဒေသလဵု၊ ပွိုင်ဍုင်လဵု မာန် အလဵုအလဵုရ။ လဒက်ပ္တန် Code ဂှ် မလေပ် ပါ်လဝ် နကဵု (၃) ကဏ္ဍ အဓိက အတိုင်ဗွဲသၟဝ်ဝွံရ -

(ဥပမာ - ပိုဲစကာ သၞောတ်ဂၞန် ၅ မ

[A]-[BB]-[CCC]

)*

ကဆံင် (၁) တွဵုရး / ဒေသဇၞော် (State/Region Level):

ဂၞန် ကိုပ်ကၠာအိုတ် (A) ဂှ် စၞောန်ထ္ၜးကဵု တွဵုရး ဟွံသေင်မ္ဂး ဒေသဇၞော်ရ။

ဥပမာ - `၁` = တွဵုရးဍုင်မန်၊ `၂` = တွဵုရးဍုင်ကရေင်၊ `၃` = ရးတၞင်သဳ။

ကဆံင် (၂) ခရိုင် / ပွိုင်ဍုင် (District/Township Level):

ဂၞန် ဒေါဝ် (BB) ဂှ် စၞောန်ထ္ၜးကဵု ပွိုင်ဍုင် ပ္ဍဲတွဵုရးဂှ်ရ။

ဥပမာ - ပ္ဍဲတွဵုရးဍုင်မန် (`၁`) မ္ဂး၊ `၀၁` = မတ်မလီု၊ `၀၂` = မုဟ်ဍုင်၊ `၀၃` = ရေဝ်။

ကဆံင် (၃) ဘာ / ဂကောံ / ညးပူဂဵု (Local Level):

ဂၞန် လက္ကရဴအိုတ် (CCC) ဂှ် စၞောန်ထ္ၜးကဵု စရင်ဘာ ဟွံသေင်မ္ဂး လၟေင်သင် ရ။

ဥပမာ - `၀၀၁` = ဘာပထမ၊ `၀၀၂` = ဘာဒုတိယ။

၀၁၅

ဂှ် ဟီုကဵုပိုဲ ဒဒှ်ရ ဍေဟ်ဒှ် ဘာလၟေင် (၁၅) ပ္ဍဲပွိုင်ဍုင်ရေဝ်။

ဂုဏ်ဖိုလ် သၞောတ်ဍာ်ဇမၠိင်

သၞောတ် Code မၞုံကဵု လဒက်ပ္တန် ဗီုဏအ်ဝွံ ယဝ်ရ ပရေင်သာသနာ ဇၞော်မောဝ်တိုန်အာ စဵုကဵု ၅၀ သၞာံ၊ ၁၀၀ သၞာံ အနာဂတ်တေအ် မ္ဂးလေဝ်၊ ပရေင်စရင်အင် ဟွံထုဲထံင်၊ လၟိဟ်သင် မုစိလ္ၚီ ဇကု မံက်ကၠုင်ကီုလေဝ် စက်ကောန်ပျူတာ (Database) တအ် စၟတ်သမ္တီ ကေုာံ သရောပ်လၟိဟ် ကဵုမာန် လောဲလောဲသွာသွာ ရ။
