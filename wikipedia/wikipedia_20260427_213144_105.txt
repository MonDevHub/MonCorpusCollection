ဝင် ကေုာံ ပရေင်ဇၞော်မောဝ်

သၞောတ် RubyGems ဂှ် စတင်ခၞံဗဒှ်တုဲ ပတိတ်သုင်စောဲ ပ္ဍဲသၞာံ ၂၀၀၄ ရ။ တင်ရန်တၟအ်အဓိကဂှ် သွက်ဂွံဖန်ကဵု သၞောတ်စွံပတန်ကုဒ် ကေုာံ ပရေင်ပါ်ပရအ်ကုဒ် အကြာသၟာချူကုဒ်ရူဗဳ (Ruby developers) ဂမၠိုင် ညံင်ဂွံလောဲသွာ ကေုာံ နွံကဵုစံင် (Standardized) ရ။ ပ္ဍဲအခိင်လၟုဟ်မ္ဂး RubyGems ဂှ် ပါလုပ်ဒၟံင် နဒဒှ်သၞောတ်သ္ဇိုင် (Built-in) ပ္ဍဲကဵု ဘာသာရူဗဳ အတိုင်ဓမ္မတာရ။

လဒက်ပတန် အပ္ဍဲ ဂျေမ်မွဲ (Structure of a Gem)

ဂျေမ်မွဲမွဲဂှ် ဗွဲမဂၠိုင် လဒက်ပတန်လဝ် နကဵု တိုက်ဝှာင် (Directories) ကေုာံ ဝှာင်အဓိက အတိုင်ဗွဲသၟဝ်ဏအ်ရ -

`gemspec` ဝှာင်:

ဒှ်ဝှာင်ကိစ္စဇၞော်အိုတ်မွဲတုဲ ဍေဟ်ဂိုင်ဒေပ်လဝ် ပရိုင်အချက်အလက် ဂျေမ်ဂှ် (ဥပမာ - ယၟုဂျေမ်၊ အဆံင် (Version)၊ ယၟုညးမချူ ကေုာံ ဂျေမ်တၞဟ် မဒးဆက်စပ်သုင်စောဲ Dependencies)။

`lib/` တိုက်ဝှာင်:

ဒှ်ဒၞာဲမစွံလဝ် ဝှာင်ကုဒ်ရူဗဳ သ္ဇိုင်အဓိက (Main Ruby code)။

`bin/` တိုက်ဝှာင်:

(ယဝ်ရနွံမ္ဂး) ဒှ်ဒၞာဲမစွံလဝ် ဝှာင်အသံစက် မလေပ်ဇက်ကမၠောန်မာန် (Executable files)။

`test/` ဟွံသေင်မ္ဂး `spec/`:

ဒၞာဲမစွံလဝ် ကုဒ်စံၜတ် (Test cases) သွက်ဂွံစၟဳစၟတ် ပရေင်ကမၠောန်ဂျေမ်။

အသံစက် မသုင်စောဲဂၠိုင်ဂမၠိုင် (Common Commands)

သၟာချူကုဒ်တအ် သုင်စောဲ အသံစက် `gem` နူကဵု Terminal သွက်ဂွံထိင်ဒဝ် ပက်ကေတ်ဂမၠိုင်ရ -

`gem install [ယၟုဂျေမ်]`: သွက်ဂွံ စုတ်/ဒါန်လုဒ် ဂျေမ်မွဲမွဲ ပ္ဍဲစက်ကောမ်ပျူတာဇကု (ဥပမာ - `gem install rails`)။

`gem update`: သွက်ဂွံ ပတိုန်အဆံင် (Update) ဂျေမ်ဂမၠိုင် စိုပ်အဆံင်တၟိအိုတ်။

`gem list`: သွက်ဂွံ ထ္ၜးစရင် ဂျေမ်အိုတ်သီု မစုတ်လဝ်ပ္ဍဲစက်။

`gem uninstall [ယၟုဂျေမ်]`: သွက်ဂွံ ဇိုတ်ပလီုပတိတ် ဂျေမ်နူစက်။

Gemfile ကေုာံ Bundler

ပ္ဍဲပရေင်ဖန်ဗဒှ် ပရဝ်ဂျေတ်ဇၞော်ဇၞော် ညံင်ရဴ ဝက်ဘ်သာ် (ဥပမာ -

Ruby on Rails

) မ္ဂး၊ ညးတအ် ဟွံသုင်စောဲ `gem install` မွဲတုဲမွဲရ။ စၞးဂှ် ညးတအ် ချူစၟတ်သမ္တီလဝ် စရင်ဂျေမ်မနွံပၟိက်အိုတ်သီု ပ္ဍဲဝှာင်တၟေင်မွဲ မနွံယၟု

`Gemfile`

ရ။ တုဲမ္ဂး ညးတအ် သုင်စောဲ ပရဝ်ဂရာမ်

Bundler

(`bundle install`) သွက်ဂွံ စုတ်ကဵု ဂျေမ်အိုတ်သီု ကေုာံ အဆံင်ဍေဟ်တအ် ညံင်ဂွံကိတ်ညဳရေင်သ္ကအ်ရ။
