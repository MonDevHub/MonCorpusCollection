ဝေါဟာ ဝါကျ

ဝေါဟာရ ဝါကျ

ဝွံ ပ္ဍဲ

သဒ္ဒါ

မန်မ္ဂး ဒှ်ဝေါဟာရ မစကာ သွက်ဂွံကော်စ ပွမပံင်ကောံလဝ် မလိက်/ဝေါဟာရဂမၠိုင် မၞုံကဵု အဓိပ္ပါယ်ဍိုက်ပေင်မွဲရ။ ပ္ဍဲအရာမလ္ၚတ် တမ်မူလဝေါဟာရ (Etymology) ကေုာံ လဒက်ပ္တန် အက္ခရဝိဒါန်မန် (Mon Orthography) မ္ဂး မလိက် "ဝါကျ" ဝွံ ဒက်ပ္တန်လဝ် နကဵု ဗျည် ကေုာံ အက္ခရ်ဖ္ဍိုဟ် မပံင်ကောံလဝ်

(ဝါ + က + ယ)

ရ။

လဒက်ပ္တန် အက္ခရဝိဒါန် (Orthographic Breakdown)

ပ္ဍဲသၞောတ် လိက်မန် ကေုာံ လဒက်ပ္တန်မလိက်မ္ဂး "ဝါကျ" ဂှ် ပါ်ထ္ၜးကေတ် နကဵု ဗျည် ကေုာံ သရ အတိုင်ဗွဲသၟဝ်ဝွံရ-

ဝါ

= ဗျည် "ဝ" + သရ "အာ" ( ာ )

ကျ

= ဗျည် "က" + အက္ခရ်ဖ္ဍိုဟ် "ယ" ( ျ - ယ-ပတ်)

ဟိုတ်ဂှ်ရ မလိက် "ကျ" ဝွံ ဒှ်ပွမပံင်ကောံလဝ် ဗျည် "က" ကေုာံ "ယ" တုဲ၊ သရုပ်လဒက်ပ္တန် မလိက်အလုံမွဲမ္ဂး

ဝါကျ = ဝါ + က + ယ

မဒှ်ရ။ နကဵုနဲကဲ ပါ်မလိက်ဏအ်ဂှ် ပသောင်ကလးကဵု ညံင်ညးမလ္ၚတ် လိက်မန်တအ် သ္ဂောံကၠိုဟ်စိုတ် ဗီုလဵု မလိက် "ကျ" ကတဵုဒှ်ကၠုင်လဝ်ဂှ်ရ။

တမ်မူလ နူ ပါဠိ ကေုာံ သက္ကတ (Etymology)

ဝေါဟာရ "ဝါကျ" ဝွံ တမ်မူလဇေတ်တ်ဂှ် ကေတ်လဝ် နူကဵု ဘာသာပါဠိ "Vākya" ကေုာံ ဘာသာသက္ကတ "Vākya" (वाक्य) ရ။ လဒက်ပ္တန် သဒ္ဒါ ပါဠိ/သက္ကတ မ္ဂး ဍေံကတဵုဒှ်ကၠုင် နူကဵု ရိုဟ်ကၞက် (Root) အတိုင်ဗွဲသၟဝ်ဝွံရ-

ဝါက် (Vāk / Vāc):

ဒှ်ရိုဟ်ကၞက် မၞုံအဓိပ္ပါယ် "အရေဝ်"၊ "ဂလာန်" ဟွံသေင်မ္ဂး "ပွမဟီုဂး" (Speech / Voice) ရ။

ယ (Ya - ဏျ ပစ္စယ်):

ဒှ်ပစ္စယ် (Suffix) မဆက်စုတ်လက္ကရဴ သွက်ဂွံ ဖန်ဗဒှ်ကဵု နာမ် ဟွံသေင်မ္ဂး အရာမဆက်စပ်ရ။

ပံင်သီုၜါမ္ဂး

Vāk + ya = Vākya

ကတဵုဒှ်ကၠုင်တုဲ၊ နွံအဓိပ္ပါယ် "အရာမဒးဒုင်ဟီုပတိတ်" ဟွံသေင်မ္ဂး "အရေဝ်ဂလာန် မဍိုက်ပေင်" ရ။ အရေဝ်မန်ပိုဲ ကေတ်စၠောအ်လဝ် မလိက်ဏအ် သီုဟွံပြံင်လှာဲ ဗီုပြင်မလိက်တုဲ ဆက်စကာကၠုင်ဒၟံင် စဵုကဵုလၟုဟ်ရ။
