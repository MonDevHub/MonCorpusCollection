သင်္ကေတ ဂစိုတ်မန်

(အင်္ဂလိက်:

Diacritic Marks for Final Consonants / Killer strokes

) ဝွံ ပ္ဍဲကဵု သဒ္ဒါမန် ကေုာံ အက္ခရ်မန်မ္ဂး ဒှ်သင်္ကေတ မစကာ သွက်ဂွံစၞောန်ထ္ၜး ဒဒှ်ရ ဗျည်မွဲမွဲဂှ် ဒှ်ဗျည်မအုပ်ကၞောတ် (ဂစိုတ်) ရ။ သင်္ကေတတအ်ဏအ်ဝွံ ပလီု ဟွံသေင်မ္ဂး ဂစိုတ်ကၠေအ် ရမျာင်သွဵု မပါလုပ်ဒၟံင် ပ္ဍဲဗျည်ဂှ်တုဲ၊ ကဵုသှ်ေဆ ရမျာင်ဗျည်သၟး သွက်ဂွံပတိတ် ရမျာင်အုပ်ကၞောတ်ရ။

ပ္ဍဲလိက်ပတ်မန်ဂှ် သင်္ကေတ မဆက်စပ် ကုဂစိုတ် အဓိက နွံ (၃) သာ် အတိုင်ဗွဲသၟဝ်ဏအ်ရ -

၁။ အသတ် ( ် )

နိဂ္ဂဟိတ်

(Niggahita / Anusvara) လဆောဝ်မ္ဂး ကော်ခဴစ "ကွက်က္ဍိုပ်" ဟွံသေင်မ္ဂး "ဗိန္ဒု" ဝွံ ဒှ်သင်္ကေတ မစွံလတူ ဗျည်တုဲ မလေပ်ပတိတ် ရမျာင် ကၟာတ်မုဟ် (Nasal final) ရ။ ဍေံဝွံ ပ္ဍဲသဒ္ဒါမန်မ္ဂး တုပ်သၟဟ် ကုရမျာင်

(Visajjani / Visarga) လဆောဝ်မ္ဂး ကော်ခဴစ "ဇုက်ၜါ" ဟွံသေင်မ္ဂး "အး" ဝွံ ဒှ်သင်္ကေတ မစွံလပါ်လက္ကရဴ ဗျည်တုဲ မလေပ်ပတိတ် ရမျာင် ကျာတိတ် (Glottal fricative final) ရ။ ဍေံဝွံ ပ္ဍဲသဒ္ဒါမန်မ္ဂး တုပ်သၟဟ် ကုရမျာင်

သင်္ကေတဂစိုတ်တအ်ဏအ်ဝွံ ကိစ္စဇၞော် ဗွဲမလောန် ပ္ဍဲအရာမပါ်ခြာ အဓိပ္ပါယ် ဝေါဟာရမန်ရ။ ယဝ်ရ ဟွံစကာ သင်္ကေတ အသတ် ( ် ) မ္ဂး၊ ဥပမာ - "ကန်" (Kon - ကောန်) ဂှ် မလေပ်ဒှ်အာ "ကန" (Kana) တုဲ အဓိပ္ပါယ် ယောင်ယာအာမာန်ရ။ ဟိုတ်ဂှ်ရ ပွမစကာ သင်္ကေတဂစိုတ် ဒးရးမ္ဂး ပကဵုညံင် လိက်မန် ဂွံတပ်တး ကၠးဖ္ဍးရ။
