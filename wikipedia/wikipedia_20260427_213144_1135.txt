သင်္ကေတစဳရေင်

သၚ်္ကေတစဳရေၚ်

(အၚ်္ဂလိက်:

Software

) ဟီုမ္ဂးဂှ် ဒှ်တၚ်နၚ်ညးစဳရေၚ် (Instructions) ဟွံသေၚ်မ္ဂး ပရဝ်ဂရမ် (Programs) မစဳရေၚ်ကဵု

ကွန်ပျူတာ

ညံၚ်ဂွံကၠောန်ကမၠောန် အတိုၚ်မနွံပၟိက်ရ။ သၚ်္ကေတစဳရေၚ်ဝွံ ဒှ်အစိတ်အပိုင်းမဟွံမွဲကဵုရုပ်ဝတ္ထု (Intangible) တုဲ ဍေံပြိုၚ်ပ္ကာန်ဒၟံၚ်ကဵု

ဟာတ်ဝဲ

(Hardware) မဒှ်ကိရိယာရုပ် မနွံကဵုရုပ်ဝတ္ထု (ဥပမာ- CPU, Monitor) ရ။

ဂကူသၚ်္ကေတစဳရေၚ်ဂမၠိုၚ်

သၚ်္ကေတစဳရေၚ်ဂှ် ပါ်ခြာလဝ် ၂ ဂကူ အဓိက:

၁. သၚ်္ကေတစဳရေၚ်သၞောတ် (System Software)

ဒှ်သၚ်္ကေတစဳရေၚ် သွက်ဂွံထိၚ်ဒဝ် ကေုာံ စဳရေၚ်ကမၠောန် ကိရိယာရုပ် (Hardware) ကေုာံ သၚ်္ကေတစဳရေၚ်တၞဟ်တအ်ရ။

သၞောတ်ဗပေၚ် (Operating System):

ဥပမာ-

အေန်ဒရွောဒ်

,

အာဲအိုအေတ်

,

Windows

,

Linux

။

သၚ်္ကေတစၠောအ်ကိရိယာ (Device Drivers):

သွက်ဂွံဆက်စၠောအ်ကဵု ကိရိယာတၞဟ် (ဥပမာ- Printer)။

၂. သၚ်္ကေတစဳရေၚ်အသုံးပြု (Application Software)

ဒှ်သၚ်္ကေတစဳရေၚ် သွက်ညးအသုံးပြုတအ် ဂွံကၠောန်ကမၠောန်မွဲမွဲ ဗွဲတၟေၚ်ရ။

သၚ်္ကေတဂၠာဲဝက် (Web Browsers):

ဥပမာ- Google Chrome, Safari သွက်ဂွံဗဵုအေန်တာနက်။

သၚ်္ကေတချူလိက် (Office Suites):

ဥပမာ- Microsoft Word, Google Docs။

ပွမခၞံဗဒှ် (Development)

သၚ်္ကေတစဳရေၚ်ဂှ် ခၞံဗဒှ်လဝ် နကဵုအရေဝ်ဘာသာသၚ်္ကေတ (Programming Languages) မပ္တံကဵု

Java

,

Python

,

C++

ရ။ ညးမခၞံဗဒှ်တအ်ဂှ် ကော်ခါစ "အေန်ဂျေန်နဳယာ သၚ်္ကေတစဳရေၚ်" (Software Engineer) ရ။

ဝၚ်ပ္ဍဲမခၞံဗဒှ်

သီအိုရဳမဆေၚ်ကဵု သၚ်္ကေတစဳရေၚ်ဂှ် စကတဵုဒှ်ကၠုၚ် နူ Alan Turing ပ္ဍဲသၞာံ ၁၉၃၅ ရ။ ဆဂး မလိက် "Software" ဂှ် စအသုံးပြုကၠုၚ် အလန်ပထမအိုတ် နကဵု John Tukey ပ္ဍဲသၞာံ ၁၉၅၈ ရ။
