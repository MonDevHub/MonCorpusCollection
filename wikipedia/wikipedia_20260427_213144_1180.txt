) ဟီုမ္ဂးဂှ် ပ္ဍဲကဏ္ဍသိပ္ပံကောမ်ပျူတာ ကေုာံ ပရေင်ချူကုဒ် (Software Engineering) မ္ဂး ဒှ်သင်္ကေတ အသံစက် (Slash Command) ဟွံသေင်မ္ဂး ဂၠံင်တရဴအေန်တာနက် (URL endpoint) မစၞောန်ထ္ၜးကဵု "လိက်စရင်သမ္တီ ကေုာံ လိက်စၞောန်လမ်း" (

Documentation

) ရ။

အလိုက်လညာတ် နဲကဲဗေဒခေတ်တၟိမ္ဂး ဝေါဟာရ "/doc" ဝွံ ဂွံဆဵုကေတ် ပ္ဍဲကဏ္ဍအဓိက (၃) သာ် အတိုင်ဗွဲသၟဝ်ဏအ်ရ -

၁. နဒဒှ် အသံစက် ပ္ဍဲ AI ကေုာံ ပရဝ်ဂရာမ်ချူကုဒ် (AI Slash Command)

ပ္ဍဲခေတ်လၟုဟ် ပရဝ်ဂရာမ်ချူကုဒ် (IDE) မပါလုပ်ကဵု

ဉာဏ်ပ္တောအ်

ညံင်ရဴ Cursor, GitHub Copilot ဟွံသေင်မ္ဂး AI Chatbot တအ်မ္ဂး၊ `/doc` ဂှ် ဒှ်အသံစက် ဗွဲမပြဟ်မွဲရ။

လၟေင်ကမၠောန်:

ကာလသၟာချူကုဒ်တအ် (Programmers) ရုဲစှ်လဝ် ကုဒ်မွဲကဏ္ဍတုဲ တက်စုတ် `/doc` မ္ဂး၊ စက် AI ဂှ် ဍေဟ်ဖန်ဗဒှ်ကဵု လိက်သောင်ကလးကုဒ် (Docstrings ဟွံသေင်မ္ဂး Comments) ဗွဲအလဵုအလဵုရ။

ဖိုလ်ဂုဏ်:

ရီုဗင်ကဵု ညံင်ဂွံချူ လိက်စၞောန်လမ်း Javadoc, ဒေါက်သထြေန် (Docstrings) ပ္ဍဲ Python ကေုာံ JavaScript တအ် ပြဟ်ပြေဟ်ရ။

၂. နဒဒှ် ဂၠံင်တရဴ Web API (API Endpoint)

ပ္ဍဲပရေင်ဖန်ဗဒှ် ဝက်ဘ် (Web Development) ကေုာံ ပရေင်စဳဇန်

API

(Application Programming Interface) မ္ဂး၊ တၠပညာတအ် လေပ်သုင်စောဲ ဂၠံင်တရဴ `/doc` ဟွံသေင်မ္ဂး `/docs` သွက်ဂွံထ္ၜး လိက်စၞောန်လမ်း သၞောတ် API ရ။
