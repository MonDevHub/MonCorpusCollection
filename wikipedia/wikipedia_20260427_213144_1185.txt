World Wide Web

(WWW) ရ။ ညးဝွံ ဒှ်ညးမပတိုန်ကသပ် ကေုာံ ဖန်ဗဒှ်လဝ် သၞောတ်

HyperText Markup Language

(HTML)၊

HTTP

ကေုာံ

URL

မဒှ်တိုင်ပကိုင်အဓိက သွက်အေန်တာနက် ခေတ်လၟုဟ်ရ။

ဘဝကိုပ်ကၠာ ကေုာံ ပညာရေး

တဳမ် ဗာနာသ်-လဳ ကတဵုဒှ်မၞိဟ် ပ္ဍဲဍုင်လန်ဒန် ရးနိဂီုအင်္ဂလာန်။ မိမညးၜါဂှ် ဒှ်တၠပညာသင်္ချာ မကမၠောန်ပ္ဍဲသၞောတ် Ferranti Mark 1 (ကောမ်ပျူတာခေတ်ကိုပ်ကၠာ) တုဲ၊ ညးဝွံ ဇၞော်မောဝ်ကၠုင် ပ္ဍဲအကြာနဲပညာကောမ်ပျူတာရ။ ညးတိုန်ဘာပ္ဍဲ အံက်သဖို့ဒ် (Queen's College, Oxford) တုဲ ဂွံဒုင်ကေတ် ဘွဲ့ရူပဗေဒ (Physics) ရ။

ပရေင်ခၞံဗဒှ် World Wide Web

ပ္ဍဲသၞာံ ၁၉၈၉ အခိင်ညးမကမၠောန်ဒၟံင် ပ္ဍဲဒၞာဲသုတေသန

CERN

ရးနိဂီုသမ္ဗာတ်သလာန်ဂှ် ညးစဆဵုကေတ် အခက်အခုဲ ပ္ဍဲပရေင်ပါ်ပရအ်တင်ဂၞန် အကြာတၠပညာသိပ္ပံဂမၠိုင်ရ။ ဟိုတ်ဂှ်ရ ညးပတိုန်ကသပ် သၞောတ်မသုင်စောဲ Hypertext သွက်ဂွံဆက်စၠောအ်တင်ဂၞန် အလုံဂၠးကဝ်ရ။

၁၉၉၀:

ညးချူပတိုန် ဝပ်ဗြောက်သာ (Web browser) ကိုပ်ကၠာအိုတ် ကေုာံ ဝပ်သာဗာ (Web server) ကိုပ်ကၠာအိုတ်။

၁၉၉၁:

မုက်လိက်ဝပ်သာဒ် ကိုပ်ကၠာအိုတ် ပ္ဍဲဂၠးကဝ် စတိုန်အာ လတူအေန်တာနက်။

၁၉၉၄:

ညးဒက်ပတန် World Wide Web Consortium (W3C) ပ္ဍဲ MIT သွက်ဂွံထိင်ဒဝ် စံကာတ်နဲပညာဝပ် (Web standards) ဂမၠိုင်ရ။

အာဲကၟာ ကေုာံ လညာတ်

တင်တၟေင် တဳမ် ဗာနာသ်-လဳ ဂှ် ညးဟွံကေတ် အခေါင်ပိုင်ပြဳ (Patent) သွက် World Wide Web ရ။ ညးရန်တၟအ်လဝ် ညံင်နဲပညာဏအ် ဂွံဒှ်အရာဗၠးၜး သွက်မၞိဟ်အလုံဂၠးကဝ် ဂွံသုင်စောဲ ဟွံဒးပေးကေတ်ၚုဟ်မးရ။ ဟိုတ်နူဂှ်ရ အေန်တာနက်ဂှ် ဇၞော်မောဝ်ကၠုင် ဗွဲမပြဟ်ရ။
