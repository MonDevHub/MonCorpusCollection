သိပ္ပံစိုတ်

ဗီုပြင် လ္ၚတ်သ္ၚဳဂၠေပ် က္ဍဟ် ကေုာံ လညာတ်စိုတ်မၞိဟ် ပ္ဍဲခေတ်တြေံ။

သိပ္ပံစိုတ်

(အင်္ဂလိက်:

Psychology

) ဟီုမ္ဂးဂှ် ဒှ်ပညာသိပ္ပံ မလ္ၚတ်သ္ၚဳဂၠေပ် ပရူ

စိုတ်

(Mind) ကေုာံ အစောံစရာဲ

ပရေင်ချဳဒရာင်

(

Behavior

) သတ်တအ်ရ။ နကဵုမတ်တၠပညာ

ယုတ္တိဗေဒ

မ္ဂး ပညာဝွံ ဒှ်သ္ဇိုင်ကိစ္စဇၞော် သွက်ဂွံကၠိုဟ်စိုတ် ပရေင်ချပ်ဗ္စာ၊ ပရေင်ဒုင်စသိုင် ကေုာံ ပရေင်ဆက်ဆောံ ဂကောံမၞိဟ်

ဇေတ်ဇေတ်

ရ။

သ္ဇိုင်လညာတ် ကေုာံ ကဏ္ဍဂမၠိုင်

သိပ္ပံစိုတ်ဝွံ လှဲလးဗွဲမလောန်တုဲ နကဵုမတ်တၠပညာမ္ဂး ပါ်ခြာလဝ်

ကဏ္ဍ

အဓိကဂမၠိုင် အတိုင်ဗွဲသၟဝ်ဏအ်ရ။

သိပ္ပံစိုတ် လညာတ်ကၠိုဟ်ခၠင် (

Cognitive Psychology

):

လ္ၚတ်ပရူ လၟေင်ကမၠောန်က္ဍဟ် ဗီုကဵု ပရေင်

စၟတ်သမ္တီ

(

Memory

)၊ ပရေင်ချပ်ဗ္စာ ကေုာံ ပရေင်သောင်ကလးပြသၞာ။

သိပ္ပံစိုတ် လွဳယဲ (

Clinical Psychology

):

လ္ၚတ်ကေုာံ လွဳပရာ ယဲစိုတ် ကေုာံ ပြသၞာပရေင်စိုတ်ဓါတ် မပ္တံကဵု ယဲစိုတ်ဟွံတန်ကြန် (

Anxiety

) ကေုာံ ယဲစိုတ်ဍိုန်လျ (

Depression

)။

သိပ္ပံစိုတ် ပရေင်ဇၞော်မောဝ် (

Developmental Psychology

):

လ္ၚတ်ပရူ ပရေင်ပြံင်လှာဲစိုတ် ကေုာံ ရုပ် နူဘဝကောန်ၚာ် စဵုကဵုစိုပ် ဘဝမၞိဟ်ဇၞော် ဗွဲမနွံကဵု ယုတ္တိဗေဒ။

သိပ္ပံစိုတ် ပရေင်မၞိဟ် (

Social Psychology

):

လ္ၚတ်သ္ၚဳဂၠေပ် ဗီုလဵု ပွဳပွူဂကောံမၞိဟ် ကေုာံ ညးတၞဟ်တအ် ဓလီုပ္ဍဲလတူ လညာတ်စိုတ် ကေုာံ စရိုတ်ပူဂိုလ်မွဲမွဲ

ဇေတ်ဇေတ်

ရ။

တင်ကိစ္စဇၞော် ပ္ဍဲပရေင်မၞိဟ်

ပညာသိပ္ပံစိုတ်ဝွံ နွံကဵု

ပရေင်သုင်စောဲ

ပ္ဍဲကမၠောန် (

Practical Application

) ဗွဲမဂၠိုင်ရ။

ပရေင်ထတ်ယုက်စိုတ် (

Mental Health

):

ဗိုင်ရီုဗင်ကဵု မၞိဟ်တအ် ညံင်ဂွံထိင်ဒဝ် ပရေင်

ထတ်ယုက်

စိုတ် ကေုာံ ဇၞော်မောဝ်ပတိုန်

ဘဝ

မနွံကဵု တသိုက်မာန်။

ပရေင်ပညာ ကေုာံ ကမၠောန်:

သုင်စောဲ နဲကဲသိပ္ပံစိုတ် သွက်ဂွံဖန်ဗဒှ် သၞောတ်ဗ္တောန်ပညာ မခိုဟ်တိုန် ကေုာံ ပရေင်စီမံခယျ မၞိဟ်ပ္ဍဲရုင်ကမၠောန် (

I/O Psychology

) ဗွဲမနွံကဵု ယုတ္တိဗေဒ ရ။

သ္ၚဳဂၠေပ် နကဵုမတ်တၠပညာ

နကဵုမတ်တၠပညာ သိပ္ပံ ကေုာံ သုတေသန မ္ဂး:

Empirical Evidence:

သိပ္ပံစိုတ်ဝွံ ဟွံသေင်ဆ လညာတ်မချပ်ကေတ်သၟး၊ ဍေဟ်ဒက်ပတန်လဝ် လတူ သက်သဳသာဓက နူကဵု ပွမစမ်ၜတ် (

Empirical evidence

) မခိုင်ကၠိုက်ရ။

Interdisciplinary Nature:

သိပ္ပံစိုတ်ဝွံ ဆက်စပ်ဒၟံင် ကုပညာတၞဟ် ဗီုကဵု ဇဳဝဗေဒ (

Biology

) ကေုာံ

လူမှုဗေဒ

(

Sociology

) တုဲ ဒက်ပတန်လဝ် ပရေင်ဆက်စပ် လညာတ် (Conceptual Continuity) ဗွဲမလှဲလး
