ဉာဏ်စက်

(AI) ကေုာံ သိပ္ပံကွန်ပျူတာ မစွံအာရီု လတူ ပွမစကာ တင်ဂၞင် (Data) ကေုာံ အယ်ဂဝ်ရဳသမ် (Algorithms) သွက်ဂွံ ဖန်ဗဒှ် ညံင်စက်ကွန်ပျူတာတအ် ဂွံဗ္တောန်ကေတ် ကေုာံ ဇၞော်မောဝ်တိုန် အလဵုဇကု ညံင်ရဴ လၟေင်ကမၠောန် ကောန်မၞိဟ်တအ် မဗ္တောန်ကေတ် ပညာကီုရ။

ပ္ဍဲကဵု သၞောတ်ကွန်ပျူတာ ဓမ္မတာမ္ဂး မၞိဟ်ချူကုဒ်တအ် သ္ဒးချူစုတ် သၞောဝ် (Rules) မွဲတုဲမွဲ သွက်ဂွံ ကဵုစက်ဂှ် ကၠောန်ကမၠောန်ရ။ ဆဂး ပ္ဍဲကဵု စက် မဗ္တောန်ကေတ် အလဵုဇကု (ML) မ္ဂး၊ စက်ဂှ် ဍေံလ္ၚတ်ကေတ် ဗီုပြင် (Patterns) နူကဵု တင်ဂၞင် လၟိဟ်ဗွဲမဂၠိုင်တုဲ၊ ဍေံအလဵုဇကုဍေံ ဖျေဟ်သ္ဂုတ်ထာဝ် ကေုာံ တော်ဆကေတ် သွဟ်မာန်ရ။

ဉာဏ်စက် မကတ်လ္ၚတ် ဉာဏ်စက်ပၠန် (AutoML / Meta-Learning)

ပ္ဍဲခေတ်တၟိဏအ်မ္ဂး ကွတ်ပညာဝွံ ဇၞော်မောဝ်တိုန် စိုပ်အာ ကဆံင်

ဉာဏ်စက် မကတ်လ္ၚတ် ဉာဏ်စက်ပၠန်

ဟွံသေင်မ္ဂး

Automated Machine Learning (AutoML)

ကေုာံ Meta-Learning ရ။

AutoML:

ဒှ်ကွတ်ပညာ မစကာ ဉာဏ်စက် (AI) မွဲ သွက်ဂွံ ကလေင်ဒက်ပ္တန် ကေုာံ ဗ္တောန်ကဵု ဉာဏ်စက် (AI) တၞဟ်မွဲပၠန်ရ။ စၞးမၞိဟ် သ္ဒးဖျေဟ် လဒက်ပ္တန် နာဗ်ကြောဂှ်၊ ဉာဏ်စက်ဂှ် ဍေံအလဵုဇကုဍေံ ရုဲစှ်ကေတ် အယ်ဂဝ်ရဳသမ် မခိုဟ်အိုတ် သွက်ဂွံ သောင်ကလး ပြဿနာရ။

Meta-Learning (Learning to Learn):

ဉာဏ်စက်ဂှ် ဍေံဗ္တောန်ကေတ် "ဗီုလဵု ဂွံဗ္တောန်ကေတ်" ရ။ ဍေံစကာ ပရေင်ဆဵုဂဗ နူကဵု ကမၠောန်တြေံတအ်တုဲ၊ ကာလဆဵု ကမၠောန်တၟိမ္ဂး ဍေံဗ္တောန်ကေတ် ပြဟ်ပြဟ် သွက်ဂွံ ကၠောန်မာန်ရ။

ဂကူ နဲကဲ မဗ္တောန်ကေတ်ဂမၠိုင် (Categories of Machine Learning)

စက် မဗ္တောန်ကေတ် အလဵုဇကု တအ်ဂှ် ပါ်လဝ် နဲကဲ အဓိက (၃) သာ်ရ-

ပွမဗ္တောန် နကဵု ပရေင်စၞောန်ပညုင် (Supervised Learning):

စက်ဂှ် ဒးဒုင်ဗ္တောန်လဝ် နကဵု တင်ဂၞင် မၞုံကဵု သွဟ် (Labeled data) ရ။ ဥပမာ- ထ္ၜးကဵု ဗီုရုပ် ဗ္ဂဲ လၟိဟ် (၁၀၀၀) ဗီု တုဲ ကဵုသမ္တီလဝ် "ဏအ်ဂှ် ဗ္ဂဲ" ရ။ ကာလထ္ၜး ဗီုရုပ် ဗ္ဂဲတၟိမွဲမ္ဂး စက်ဂှ် စၟတ်သမ္တီမာန်ရ။

ပွမဗ္တောန် အလဵုဇကု (Unsupervised Learning):

စက်ဂှ် ကလိဂွံ တင်ဂၞင် မဟွံမွဲကဵု သွဟ် (Unlabeled data) တုဲ၊ ဍေံအလဵုဇကုဍေံ သ္ဒးဂၠာဲ ဗီုပြင် (Patterns) ကေုာံ ဒက်ဂကောံ တင်ဂၞင် (Clustering) အလဵုဇကုရ။ (ဥပမာ- ပွမပါ်ကရေက် ဂကောံညးရာန်ကပေါတ် ပ္ဍဲကဵု ပရေင်ပိုန်ဒြပ်)။

ပွမဗ္တောန် နကဵု ပရေင်ဆဵုဂဗ / ဒဏ် (Reinforcement Learning):

စက်ဂှ် ဍေံဗ္တောန်ကေတ် နကဵု လၟေင်ကမၠောန် စမ်ၜတ် ကေုာံ ယောင်ယာ (Trial and error) ရ။ ယဝ်ဍေံကၠောန်ဒးမ္ဂး ကလိဂွံ လာပ် (Reward)၊ ယဝ်ကၠောန်ဗၠေတ်မ္ဂး ဒးဒုင်ဒဏ် (Penalty) ရ။ ဉာဏ်စက် AlphaGo မဝေင်ဇၞး မၞိဟ် ပ္ဍဲကဵု ဂိမ်း Go ဂှ် စကာလဝ် နဲကဲဏအ်ရ။

ကွတ်ပညာ မဆက်စပ် ကဆံင်သၠုင် (Advanced Subfields)

ပွမဗ္တောန် ကဆံင်ဇြိုဟ် (Deep Learning):

ဒှ်ကဏ္ဍခွဲ မၞုံပ္ဍဲ Machine Learning ပၠန်တုဲ၊ ဍေံစကာ သၞောတ် ကွန်ရက် နာဗ်ကြော အတု (Artificial Neural Networks) မၞုံကဵု ထပ် (Layers) ဗွဲမဂၠိုင် သွက်ဂွံ သ္ၚေဝ်ဂၠိပ် တင်ဂၞင် မဇြိုင်သ္ကာတ် ညံင်ရဴ က္ဍဟ်ကောန်မၞိဟ်ကီုရ။

ပရေင်လ္ၚတ် အရေဝ်ဘာသာ သဘာဝ (Natural Language Processing - NLP):

ပွမစကာ Machine Learning သွက်ဂွံ ဖန်ကဵု ညံင်စက် ဂွံကၠိုဟ်ခၠင် ကေုာံ ဟီု အရေဝ်ဘာသာ ကောန်မၞိဟ်မာန် (ဥပမာ- ChatGPT, Google Translate)။

အကျိုဟ် ကေုာံ ဒၞာဲမစကာ (Applications)

စက် မဗ္တောန်ကေတ် အလဵုဇကု ဝွံ လုပ်ပါလုပ်ဒၟံင် ပ္ဍဲကဵု ဘဝလၟေင်တ္ၚဲ ပိုဲတအ် ဗွဲမဂၠိုင်ရ-

သၞောတ် ထံက်ထ္ၜး (Recommendation Systems):

ဝှုက်ဗောက် (Facebook)၊ ယူကျု (YouTube) ကေုာံ Netflix တအ် စကာ ML သွက်ဂွံ တီကေတ် စိုတ်ဓါတ် ညးလွပ်တုဲ ထံက်ထ္ၜးကဵု ဗဳဒဳယဝ် ဟွံသေင်မ္ဂး ကြော်ငြာ မကိတ်ညဳ ကုမၞးရ။

ပရေင်ထတ်ယုက် ကေုာံ ဂဥုဲ (Healthcare):

စကာ ML သွက်ဂွံ စၟဳစၟတ် ဗီုဓါတ် X-ray တုဲ ဂၠာဲယဲကေန်သာ (Cancer) မာန် ပြဟ်နူ အစာသဝ်မၞိဟ်ဏီရ။

ကွဳစက် ဟွံမွဲကဵုမၞိဟ်ဒဳ (Autonomous Vehicles):

ကွဳစက် တက်သလာ (Tesla) တအ် စကာ ML သွက်ဂွံ ညာတ်ကေတ် ကွဳစက်တၞဟ်၊ မၞိဟ် ကေုာံ လိက်စၞောန် ဂၠံင်တအ်တုဲ ဒဳအာ အလဵုဇကုမာန်ရ။
