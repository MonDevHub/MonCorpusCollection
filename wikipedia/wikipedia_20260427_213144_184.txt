World Wide Web

; ကော်ဂစှ်ေ

ဟွံသေင်မ္ဂး

The Web

) မ္ဂးဂှ် ဒှ်သၞောတ်တင်ဂၞန် (Information system) မွဲ မကဵုအခေါင် သွက်ဂွံဍဵုဗှ်လ္ၚတ် မုက်လိက်ဝေပ် ကေုာံ တင်ဂၞန်ဒဳဂျဳတောလ်နာနာ နကဵုနဲကဲ

အေန်တာနက်

ရ။ ဇာသၠဲဂၠးဝွံ ဒုင်သဇိုင်လဝ် ကုနဲကဲ ဟိုက်ပါတေတ် (Hypertext) မဒှ်နဲကဲမဆက်စပ်လဝ် မုက်လိက်မွဲ ကုမွဲ နကဵုလေန် (Links) ဂမၠိုင်ရ။

တင်တၟေင် အကြာ အေန်တာနက် ကေုာံ ဇာသၠဲဂၠး

မၞိဟ်ဗွဲမဂၠိုင် ထေင်ခယျလဝ် ဒဒှ်ရ အေန်တာနက် ကေုာံ ဇာသၠဲဂၠးဂှ် ဒှ်အရာတူရေင်သကအ်ရ။ ဆဂး ဍေဟ်တအ် တၞဟ်ခြာဒၟံင်-

အေန်တာနက်

:

ဒှ်သၞောတ်ဇှေတ်ဝေတ် (Hardware infrastructure) မတွံဂး ကြေပ်ဂၠံင်၊ သေဝ်ဗာ ကေုာံ ကိရိယာမဆက်စပ်လဝ် အလုံဂၠးကဝ်။

ဇာသၠဲဂၠး (WWW):

ဒှ်ဝန်ကရဏ (Service) မွဲ မနွံဒၟံင် ပ္ဍဲလတူအေန်တာနက် ညံင်ရဴ ဝေပ်သာ်ဂမၠိုင်ရ။

ဝင် (History)

ဇာသၠဲဂၠးဝွံ စတင်ခၞံဗဒှ်လဝ် နကဵုတၠပညာအင်္ဂလိက်

တင် ဘာနာလိ (Tim Berners-Lee)

ပ္ဍဲသၞာံ ၁၉၈၉ အခိင်ညးမကၠောန်ကမၠောန်ဒၟံင် ပ္ဍဲဌာနသုတေသန CERN ရ။ တင်ရန်တၟအ်အဓိကညးဂှ် သွက်ဂွံဖန်ဗဒှ်ကဵု နဲကဲမွဲ ညံင်တၠပညာသိပ္ပံဂမၠိုင် ဂွံဖလှာဲတင်ဂၞန်ရေင်သကအ် လောဲသွာရ။

နဲကဲသ္ဇိုင်ဂမၠိုင် (Core Technologies)

ဇာသၠဲဂၠးဝွံ ဒုင်သဇိုင်လဝ် ကုနဲကဲ (၃) သာ် အဓိက-

HTML (Hypertext Markup Language):

အရေဝ်ဘာသာ သွက်ဂွံချူဒက်ပတန် မုက်လိက်ဝေပ်။

HTTP (Hypertext Transfer Protocol):

သၞောတ်သၞောဝ် သွက်ဂွံပလံင်ဗစး တင်ဂၞန် အကြာဗြောက်သာ ကေုာံ သေဝ်ဗာ။

URL (Uniform Resource Locator):

ယၟုဒတန် မဗၟံက်ထ္ၜး ဒၞာဲဒတန် မုက်လိက်မွဲမွဲ။

အစောံသြဇာ

ဇာသၠဲဂၠးဝွံ ပြံင်လှာဲထောအ် ဗီုပြင်မၞိဟ်တအ် ဂၠာဲဗဟုသုတ၊ ကၠောန်ပရေင်ပိုန်ဒြပ် ကေုာံ ဆက်ဆောံရေင်သကအ်ရ။ မပ္တံကဵု ဝေပ်သာ်

ဝဳကဳပဳဒဳယာ

၊ Google ကေုာံ Facebook တအ်ဝွံ ကတဵုဒှ်ကၠုင်လဝ် ဟိုတ်နူသၞောတ်ဇာသၠဲဂၠးဏအ်ရ။
