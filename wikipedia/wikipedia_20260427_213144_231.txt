ဗီုပြင် တိဗ္ၚ ကေုာံ ဒၞာဲဒတန် ဍုင်ပံင် ပ္ဍဲတွဵုရးဍုင်မန်။

ဍုင်ပံင်

(အင်္ဂလိက်:

Paung

) ဟီုမ္ဂးဂှ် ဒှ်ဍုင်မွဲ မနွံပ္ဍဲ

ခရိုင်သဓီု

၊

တွဵုရးဍုင်မန်

၊ ကၟိန်ဍုင်ဗၟာရ။ နကဵုမတ်တၠပညာ

ပထဝဳဗေဒ

မ္ဂး ဍုင်ပံင်ဝွံ နွံဒၟံင် ပ္ဍဲဒၞာဲသ္ဇိုင် ဗျူဟာကိစ္စဇၞော် မဆက်စၠောအ်လဝ် အကြာဍုင်သဓီု ကေုာံ ဍုင်မတ်မလီု တုဲ ဒှ်ဒေသတိုက်ဗ္ၚ မနွံကဵု

ယုတ္တိဗေဒ

ပရေင်ပိုန်ဒြပ်

ဇေတ်ဇေတ်

ရ။

ဒတန် ပထဝဳဗေဒ ကေုာံ ပရေင်အုပ်ဓုပ်

ဍုင်ပံင်ဝွံ ဒက်ပတန်လဝ် နကဵုကွာန်ဂမၠိုင် မနွံဗွဲမဂၠိုင်တုဲ၊ နကဵုပထဝဳမ္ဂး နွံဒၟံင်သၟဝ်

ဒဵုဂြိုပ်မတ္တမ

ရ။

ဒၞာဲဒတန်:

လပါ်ပၠေင် (အရှေ့) နွံဒဵု၊ လပါ်သၟေဝ် (အနောက်) နွံၜဳမတ္တမ (

Gulf of Martaban

)။

ပရေင်အုပ်ဓုပ် (

Administration

):

ဍုင်ပံင်ဝွံ ပါ်ခြာလဝ် နကဵုရပ်ကွက်ဂမၠိုင် ကေုာံ ကွာန်ဂမၠိုင် (ဥပမာ - ကွာန်ရင်းငြိမ်၊ ကွာန်ကျွဲခြံ) ဗွဲမနွံကဵု သၞောတ်အစာံ (Standardization) ရ။

ပရေင်ပိုန်ဒြပ် ကေုာံ သဘာဝ (

Economy

)

ဒြဟတ်ပရေင်ပိုန်ဒြပ် ဍုင်ပံင်ဂှ် တန်တဴဒၟံင် လတူကမၠောန်ဗ္ၚ ကေုာံ ပရေင်တဵုလွဳရ။

ကမၠောန်ဗ္ၚ:

ဍုင်ပံင်ဝွံ ဒှ်ဒေသ ပ္တိတ်သ္ၚု အဓိကမွဲ ပ္ဍဲတွဵုရးဍုင်မန် ဟိုတ်နူနွံကဵု တိဗ္ၚလှဲလး (

Alluvial plain

)။

ပရေင်တဵုလွဳ:

တဵုလွဳ ရဝ်ဗါ (Rubber) ကေုာံ သတ်ဆုသတ်ဒြာဲနာနာ (ဥပမာ - ဒူးရင်း၊ ကြက်မောက်)။

ပရေင်စက်မှု (

SME

):

ကမၠောန်စက်သ္ၚု ကေုာံ ကမၠောန်တၟအ် (

Quarrying

) ပ္ဍဲဒဵုဇိုင်ဍုင်ပံင်တအ်ဂှ် ကဵုဖဵု ကုပရေင်ပိုန်ဒြပ်ဒေသ

ဇေတ်ဇေတ်

ရ။

ရာဇဝင် ကေုာံ ယေန်သၞာင် (

History & Culture

)

ဍုင်ပံင်ဝွံ နွံကဵု ရာဇဝင်မဇြိုဟ်နက်တုဲ ဆက်စပ်ဒၟံင် ကု

ရာဇဝင်

ဍုင်မတ္တမ ကေုာံ ဍုင်သဓီုရ။

ဂကူ ကေုာံ အရေဝ်ဘာသာ:

ဂကူမန် ကေုာံ ဂကူဗၟာ မံင်တဴမွဲစွံတုဲ၊ ပရေင်ဂျိုင်တန်တဴ အရေဝ်ဘာသာ (

Survival of Language

) မန်ဂှ် ပ္ဍဲကွာန်ဇိုင်ဒဵုတအ် ဆက်နွံဒၟံင် ဗွဲမနွံကဵု ယုတ္တိဗေဒ ရ။

ဘာလကျာ် ကေုာံ စေတဳ:

စေတဳယၟုမြဟ်ဂမၠိုင် ကေုာံ ဘာလကျာ်တြေံတအ်ဂှ် ထ္ၜးကဵု အစောံစရာဲ

ယေန်သၞာင်

ဗုဒ္ဓဘာသာ မခိုင်ကၠိုက်ရ။

သ္ၚဳဂၠေပ် နကဵုမတ်တၠပညာ

နကဵုမတ်တၠပညာ

ဗေဒပရေင်မၞိဟ်

(

Sociology

) မ္ဂး:

Urbanization (ပရေင်ဇၞော်မောဝ်ဍုင်):

ဍုင်ပံင်ဝွံ ပြံင်လှာဲကၠုင် နူဒေသကွာန်ဗ္ၚ ဇရေင်ဒေသဗၞိဟ် (Trading hub) ဟိုတ်နူ ဂၠံင်ကဏ္ဍတုဲ (

Infrastructure development

) မနွံကဵု လၟေင်လၟာ် (Hierarchy) အဆံင်ဍုင်ရ။
