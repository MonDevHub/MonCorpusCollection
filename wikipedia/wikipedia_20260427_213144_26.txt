ကွေန် (CU) သယူ

(အင်္ဂလိက်:

The Control of Memory / Conscious Retention / Memory Processing Unit

) ဝွံ ဒှ်ဝေါဟာရ

တော်ကော်

မန် မစၞောန်ထ္ၜးကု အစောံစရာဲစိုတ် ကေုာံ သၞောတ်အာရီု မမာန်ထိင်ဒဝ်၊ ကောပ်ကာဲ ကေုာံ သိုပ်စွံလဝ် သမ္တီလက္ကရဴ (Memories) ဂမၠိုင်ရ။ ဝေါဟာရဏအ်ဝွံ ပံင်ဗစပ်လဝ် အစောံစရာဲမဖျေံအမိင် (ကွေန်) ကေုာံ အစောံစရာဲမမင်မဲလဝ်တင်ဂၞင် (သယူ) တုဲ ဍေံဒှ်သဇိုင်အဓိက သွက်ဂွံဒက်ပ္တန်

လျးညာဏ်

မၞိဟ်ရ။

လဒက်ပ္တန် ကေုာံ အဓိပ္ပါယ်

ဝေါဟာရဏအ်ဝွံ ဒက်ပ္တန်လဝ် နကဵုသဘဴဓရ် (၂) ပါ် အတိုင်ဗွဲသၟဝ်ဏအ်ရ -

ကွေန် (Control Unit - CU):

မဂွံအဓိပ္ပါယ် ဗဟဵုမကောပ်ကာဲ၊ မဖျေံလၟေင် ကေုာံ မဖျေံသ္ဂုတ်သွာတ်။ ပ္ဍဲဒၞာဲဏအ် ဍေံယိုက်ဂၠေင်တာလျိုင် သွက်ဂွံရုဲစှ် တင်ဂၞင်လဵု မထေက်ကဵုဒးသမ္တီ၊ တင်ဂၞင်လဵု မထေက်ကဵုဒးဝိုတ်။

သယူ (Memory / Suyu):

မဂွံအဓိပ္ပါယ် ပွမသိုပ်စွံ၊ ပွမမင်မဲလဝ် ဟွံသေင်မ္ဂး သမ္တီလက္ကရဴ။ ဍေံဒှ်ဒၞာဲမစွံလဝ် ပရူပရာ၊ ရသ ကေုာံ ပရေင်ဆဵုဂဗဘဝ သီုဖအိုတ်။

ပွမဒုင်တဲ (Input):

အာရီု (၅) သာ် (မပ္တံကု

အာရီုသၟူ

) ဗလးဏာ တင်ဂၞင် ဇရေင်ဗဟဵု။

ပွမကောပ်ကာဲ (Processing/Control):

"ကွေန်" သ္ၚေဝ်ဂၠေပ်ရံင် တင်ဂၞင်ဂှ် မၞုံကဵုရသစိုတ်ဓာတ် ကွေဟ်ဟာ? ယဝ်ရဍိုက်ပေင်ကု ရသစိုတ် (ဥပမာ - သၟူမြမောဝ် NAXOS) မ္ဂး "ကွေန်" ဖျေံအမိင် ညံင်ဂွံသိုပ်စွံလဝ် ဗွဲမခိုင်ကၠိုက်။

ပွမသိုပ်စွံ (Storage):

တင်ဂၞင်ဂှ် လုပ်စိုပ်အာ "သယူ" တုဲ ဒှ်အာသမ္တီလက္ကရဴ မသ္ၚောဲ (Long-term Memory)။

ပွမကလေင်ကော် (Recall):

ကာလဆဵုဂဗကု
