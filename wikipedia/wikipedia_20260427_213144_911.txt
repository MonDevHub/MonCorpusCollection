အရီုအဗင် (Help)

ဝွံ ဒှ်ဒၞာဲသွက်ဂွံလ္ၚတ် နဲကဲနာနာ သွက်ဂွံချူ ကေုာံ ပလေဝ်လိက် ပ္ဍဲဝဳကဳပဳဒဳယာ ဘာသာမန် ရ။ ယဝ်ရမၞး ဒှ်ညးစကၠုင်တၟိမွဲမ္ဂး လ္ၚတ်ကေတ် နူတင်ဂၞင် သၟဝ်ဏအ်မာန်ရ။

သွက်ညးစကၠုင်တၟိ

နိဒါန်

: ဝဳကဳပဳဒဳယာဂှ် မုရော? ဂွံစကၠောန် ဗီုလဵုရော?

နဲကဲပလေဝ်လိက်

: နဲကဲစကာ VisualEditor ကေုာံ Wikitext။

ပွမဖန်ဗဒှ်အကံက်

: မုဟိုတ်ဒးနွံ အကံက်ရော?

နဲကဲပလေဝ်လိက် (Editing Basics)

ဝဳကဳမာတ်အပ် (Wikitext)

: နဲကဲချူလိက် နကဵု Code ဂမၠိုင်။

ပွမစုတ်နိဿဲ

: နဲကဲထ္ၜးသက်သဳ သွက်တင်ဂၞင်မချူလဝ်။

နဲကဲစုတ်ဗီုရုပ်

: နဲကဲပတိုန်ရုပ် (Upload) ကေုာံ နဲကဲစုတ်ရုပ် ပ္ဍဲဆောင်းပါး။

မူဝါဒ ကေုာံ တင်စၞောန်

မူဝါဒ ၅ သာ်

: တင်ဒုင်သဇိုင် ဝဳကဳပဳဒဳယာ။

စွံစိုတ်လပါ်မဗၠးၜး

: နဲကဲချူလိက် နကဵုစိုတ်ဓါတ်မဗၠးၜး။

တင်စၞောန်ဂမၠိုင်

: တင်စၞောန် သွက်ဂွံချူလိက်ပရေင် ညံင်ဂွံခိုင်ကၠိုက်။

အဆက်အစပ် ကေုာံ ပရေင်ကောံဓရီု

သုဓမ္မာ (Village pump)

: ဒၞာဲသွက်ဂွံသၟာန် တင်သံသယ ကေုာံ ဓရီုကျာ ပရေင်ဝဳကဳမန်။

မုက်လိက်ဓရီုကျာ

: နဲကဲဓရီုကျာ ကုညးပလေဝ်လိက်တၞဟ်တအ်။

နဲကဲတြးပ္တိတ် (Advanced Editing)

ထာမ်ပလိက် (Templates)

: နဲကဲစကာ ကေုာံ ဖန်ဗဒှ် ထာမ်ပလိက်။

ကဏ္ဍ (Categories)

: နဲကဲပကောံစွံ ဆောင်းပါးဂမၠိုင်။
